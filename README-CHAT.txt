Txat berria inplementatuta dago TPV proiektuan.

Erabilera:
1. Abiarazi Java zerbitzaria: ChatSeguruJava proiektuko ChatZerbitzaria.
2. TPV ireki eta menuan sartu (sukaldaria edo zerbitzaria).
3. Txata automatikoki konektatzen da 127.0.0.1:5555 helbidera.

Aldaketak:
- Aurreko JSON bidezko txata kendu eta ordezkatu da.
- Orain Java zerbitzariarekin bateragarria da.
- SSL/TLS erabiltzen du.
- Mezuak AES/GCM bidez zifratzen dira.
- Zerbitzariaren ziurtagiria TPV/ziurtagiriak/server.crt fitxategian sartu da.
